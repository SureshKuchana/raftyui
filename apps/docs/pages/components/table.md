# Import

Table component is used to organize and display data efficiently. It renders a `<table>` element by default

- Table
- TableBody
- TableContainer
- TableFooter
- TableHead
- Td
- Th
- Tr

```jsx
import { Table, TableBody, TableContainer, TableFooter, TableHead, Td, Th, Tr } from "@rafty/ui";
```
